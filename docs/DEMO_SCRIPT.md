# 影鉴 Aegis Review 8 分钟演示稿

总时长严格为 480 秒。演示使用 Docker 正式服务、真实训练模型和已生成的
图片/视频任务，不临时训练模型。

## 0:00–0:45 背景与目标（45 秒）

- 介绍方向 A：游戏数字媒体内容的自动审核与人工复核。
- 点出痛点：单次检测结果不足以支撑审核，需要状态、证据、规则、改判和导出闭环。
- 展示产品名“影鉴 Aegis Review”和完整流程：
  上传 → 异步分析 → 三档结论 → 证据 → 人工改判 → 历史 → 导出。

## 0:45–1:45 系统架构（60 秒）

- Flask 模块化单体，单进程 `JobService` 与单 worker 线程池。
- JSON 与本地目录持久化，随机临时文件、文件 fsync、原子替换和路径白名单。
- OpenCV/FFmpeg 负责媒体，Ultralytics YOLO 负责五类检测，纯函数规则产生
  pass/review/reject。
- Docker 是正式验收路径，Conda 是宿主机回归路径。

## 1:45–5:45 完整业务流程（240 秒）

1. **上传与状态（45 秒）**  
   上传 `sample_5s.mp4`，说明 created、queued、running、completed；展示
   `analysis_in_progress.png` 或现场轮询。

   下的三档结论与边界值。
3. **证据与视频（45 秒）**  
   展示检测框、类别、置信度、时间戳、视频证据帧和检测汇总。
4. **人工改判（40 秒）**
