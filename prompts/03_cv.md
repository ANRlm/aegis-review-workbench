## 固定接口

```python
class Detector:
    def detect(self, frame: numpy.ndarray, confidence: float) -> list[dict]:
        ...

def analyze_asset(
    input_path: pathlib.Path,
    evidence_dir: pathlib.Path,
    result_dir: pathlib.Path,
    settings: AuditSettings,
    detector: Detector,
) -> AnalysisReport:
    ...
```

Detection 字段固定为 `frame_index/timestamp_seconds/class_id/class_name/confidence/bbox_xyxy/evidence_file`。输出只能含 JSON 原生类型。

## 分步工作

1. 迁移 96/24 数据与标签，记录 Day05/Day06 来源；先写数据配对、类别和坐标失败测试。
2. 编写确定性数据校验脚本；`data.yaml` 使用项目相对路径。
3. 先写 `0.35/0.60/0.20/0.80` 审核边界测试，再实现纯规则函数。
4. 先写图片、变 FPS、采样上限和损坏视频测试，再实现 sampler。
5. 实现 Ultralytics Detector 适配器，单模型缓存，Tensor/NumPy 数值立即转 Python。
6. 先用假 Detector 写 pipeline 测试，再生成证据、JSON、CSV 和 ZIP。
7. 无目标时保存第一帧；模型缺失和解码失败必须抛可读异常。
8. 提供 30 epoch、640、batch 2、CPU、seed 20260718 的训练入口。
9. 训练后只提交 `aegis_game_best.pt`、指标、必要曲线和 SHA-256。
10. 写模型文档：数据来源、类别、参数、指标、限制和重训命令。

规则顺序：高风险 `>=0.60` 拒绝；`[0.35,0.60)` 待复核；低置信度风险帧比例严格在 `(0.20,0.80)` 待复核；否则通过。
